後記：Linux 悲歌

首先，不得不說我非常不喜歡用 LINE，
要不是跟現實妥協我還真的是不會用它，連碰都不想碰。

再來，消費者面向的軟體在 Linux 非常非常缺乏，
要不然就是，雖然有但是程式可用的功能跟 Windows/macOS 差異頗大。

舉例來說，
Microsoft Office？沒上（只有上 Teams 而且還是 Beta？！）
Edge Chromium？還沒上（到底要什麼時候才上啊？）
還有本篇的 LINE。

只能說這真的是 Linux 悲歌了吧…。
