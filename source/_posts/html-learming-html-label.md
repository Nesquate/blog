以下統整可以使用的HTML標籤及屬性筆記，往後文章依舊。

### <html>標籤

標籤

說明

lang

表示網頁的所使用的語系。 在Chrome中，網頁語言與當前語言不同時，會跳出翻譯工具， 可做為該程式碼的測試現象。

```
<html lang="zh">
</html>
```

### <meta>標籤

標籤

說明

charset

表示網頁所使用的字元集。

```
<meta charset="UTF-8">
```

name content

name表示要描述的網頁特性項目； content表示此項目的內容說明。 SEO時常見。

```
<meta name"keywords" content="PC,Hardware">
```
